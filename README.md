# Trend POS - نظام ترند المتكامل

نظام إدارة المبيعات والمخزون والمحاسبة المتكامل، مصمم ليعمل بكفاءة عالية مع دعم كامل للعمل بدون إنترنت (PWA).

## المميزات الرئيسية
- **إدارة الموظفين:** نظام متكامل لإدارة بيانات الموظفين، الرواتب، وإحصائيات الأداء.
- **إدارة المخزون:** تتبع المنتجات، الكميات، والتنبيهات عند نقص المخزون.
- **المبيعات والمشتريات:** تسجيل العمليات المالية وإصدار الفواتير.
- **المحاسبة:** إدارة الصناديق النقدية، التحويلات بين العملات، والتقارير المالية.
- **دعم الأوفلاين (PWA):** يعمل التطبيق بدون إنترنت مع إمكانية التثبيت على أجهزة الهاتف والكمبيوتر.
- **واجهة عصرية:** تصميم "Glassmorphism" مع دعم كامل للغة العربية.

## التقنيات المستخدمة
- **Frontend:** React, Vite, Tailwind CSS, Framer Motion, Lucide Icons.
- **Backend:** Node.js, Express, SQLite (Better-SQLite3).
- **PWA:** Vite PWA Plugin.

## كيفية التشغيل محلياً
1. قم بتحميل المشروع.
2. قم بتثبيت التبعيات: `npm install`
3. قم بتشغيل الخادم: `npm run dev`

## GitHub Integration
لرفع المشروع على GitHub:
1. قم بإنشاء مستودع (Repository) جديد على GitHub.
2. قم بتنفيذ الأوامر التالية في مجلد المشروع:
   ```bash
   git init
   git add .
   git commit -m "Initial commit"
   git branch -M main
   git remote add origin [رابط_المستودع_الخاص_بك]
   git push -u origin main
   ```

## PWA & APK
- التطبيق يدعم خاصية PWA، مما يعني أنه يمكنك "تثبيته" مباشرة من المتصفح على هاتفك ليظهر كأيقونة تطبيق (APK).
- عند فتح التطبيق في المتصفح (Chrome أو Safari)، اختر "إضافة إلى الشاشة الرئيسية" (Add to Home Screen).
